# i2i-Academy-IntroductionToCloud-1

This project demonstrates setting up a VM on Google Cloud Platform (GCP), 
performing a ping test, connecting via SSH, and creating a remote file.

## Tasks Completed
- Created a VM instance on GCP (us-central1)
- Configured firewall rules to allow ICMP and SSH traffic
- Performed a successful ping test to the VM's public IP
- Connected to the VM via SSH
- Created hello.txt file containing "Hello i2i Academy!"
